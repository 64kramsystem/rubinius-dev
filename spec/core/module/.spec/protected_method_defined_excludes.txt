Module#protected_method_defined? returns true if the named protected method is defined by module or its ancestors
Module#protected_method_defined? should return false if method is not a protected method
Module#protected_method_defined? returns false if the named protected method is not defined by the module or its ancestors
Module#protected_method_defined? should except symbols as a method name
Module#protected_method_defined? raise an error on improper argument
