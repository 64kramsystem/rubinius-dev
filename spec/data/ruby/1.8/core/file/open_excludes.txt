File.open opens a file with a file descriptor d and a block
File.open raises an IO exception when write in a block opened with RDONLY mode
File.open raises an IO exception when write in a block opened with 'r' mode
File.open raises an IO exception when read in a block opened with File::RDONLY|File::APPEND mode
File.open can't read in a block when call open with File::EXCL mode
File.open raises an Errorno::EEXIST if the file exists when open with File::RDONLY|File::APPEND
File.open can't write in a block when call open with File::TRUNC mode
File.open raises an Errorno::EEXIST if the file exists when open with File::RDONLY|File::TRUNC
File.open opens a file for read-write and truncate the file
File.open opens a file for binary read-write and truncate the file
