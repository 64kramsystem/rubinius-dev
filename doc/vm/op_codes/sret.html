sret

Byte code: 0x6a

Operation

Simple return from a method (only)

Format

sret

Stack

Before After
retval
...
 =>
...

Description

Pops the top value from the stack, and uses it as the return value from a method.

Notes

sret is an optimised version of the more general ret. It works only with method (MethodContext) returns, but as a result, can skip the extra work to figure out how to long return from a block.
