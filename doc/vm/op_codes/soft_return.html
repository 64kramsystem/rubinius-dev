soft_return

Byte code: 0x2d

Operation

Soft return from a block

Format

soft_return

Stack

Before After
retval
...
 =>
...

Description

Pops the return value from the stack, and returns to the calling method or block. The return value is pushed onto the stack of the caller during the return.

Notes

Unlike ret, this return opcode does not consider non-local returns. It simply returns to the calling block or method context. Thus, it is used when, for example, breaking from a loop, or upon the normal completion of a block.
