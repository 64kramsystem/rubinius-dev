kind_of

Byte code: 0x5a

Operation

Evaluate if object_ is an instance of _class_ or of an ancestor of _class.

Format

kind_of

Stack

Before After
object
class
...
 =>
result
...

Description

Evaluate if the object is created by class, its parent, or a further ancestor. This differs from instance_of in that the class heirarchy will be evaluated rather than just the class object given.
