set_ivar

Byte code: 0x15

Operation

Sets an instance variable on self

Format

set_ivar ivar

Stack

Before After
value
...
 =>
...

Description

Pops a value off the stack, and uses it to set the value of the instance variable identifies by the literal ivar on the current self object.

Source

next_int;
t1 = tuple_at(state, cpu_current_literals(state, c), _int);
t2 = stack_pop();
object_set_ivar(state, c->self, t1, t2);
stack_push(t2);
