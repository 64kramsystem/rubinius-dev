open_module_under

Byte code: 0x21

Operation

Creates or opens a module nested under another module

Format

open_module_under name

Stack

Before After
parent
...
 =>
module
...

Description

Pops an enclosing class or module from the stack, and creates or re-opens the module identified by the literal _name_. The module is then pushed onto the stack.

Example

module X::Y    # Opens module Y under the module X
end
