deallocate_stack

Byte code: 0x62

Operation

Removes a specified number of items from the stack

Format

deallocate_stack count

Description

Removes the number of items specified in count from the stack.

Notes

This opcode simply decrements the stack pointer; hence the allocate_stack opcode initializes newly allocated slots to nil, to ensure old values are cleared.
