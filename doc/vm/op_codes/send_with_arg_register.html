send_with_arg_register

Byte code: 0x3d

Operation

Sends a message with args to a receiver

Format

send_with_arg_register methodname

Stack

Before After
recv
args
...
 =>
...

Description

Pops an object (recv) and an array of arguments (args) off the top of the stack, and sends the object the message methodname.

Notes

The args register must have previously been set to the count of the number of arguments in args.
