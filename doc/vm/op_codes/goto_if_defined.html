goto_if_defined (gid)

Byte code: 0x17

Operation

Jump execution to the position specified by the label if the top of the stack is not undefined.

Format

goto_if_defined label

Stack

Before After
value
...
 =>
...

Description

Remove the top item on the stack, and if it is not the special value undefined, jump to the instruction following the specified label; otherwise, continue.
