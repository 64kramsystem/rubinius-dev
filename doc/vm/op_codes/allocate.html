allocate

Byte code: 0x4

Operation

Allocate space for a new object

Format

allocate

Stack

Before After
count
...
 =>
object
...

Description

Consume a field count and allocate space for an object with the appropriate header and field space. The object created will be uninitialized; it will have its parent class and all fields initialized to the special value nil.

Source

_lit = stack_pop();
