store_my_field

Byte code: 0x41

Operation

Store a value into a field of 'self'

Format

store_my_field <arg1>

Stack

Before After
value
...
 =>
value
...

Description

Stores the value at the top of the stack into the field specified by arg1.

The stack is left unmodified.

Source

next_int;
SET_FIELD(c->self, _int, stack_top());
