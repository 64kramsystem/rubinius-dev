open_module

Byte code: 0x20

Operation

Creates or reopens a module

Format

open_module mod

Stack

Before After
...
 =>
module
...

Description

Creates or re-opens the module referenced by the literal _mod_, and pushes it onto the stack.

Example

module A   # Creates module A (or re-opens it if it already exists)
end
