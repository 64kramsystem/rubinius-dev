from_fp

Byte code: 0x5e

Operation

Push an object from the caller's stack onto the current context's stack

Format

from_fp idx

Stack

Before After
...
 =>
value
...

Description

Copies a method argument that was on the stack before the method call and pushes it onto the stack in the current context. The argument idx counts backwards from the frame pointer, with 0 referencing the first argument to the mehod, 1 the second argument, etc.

Notes

The opcode from_fp exists as an optimisation in cases where arguments to a method are not modified. Rather than copying the argument off the stack and into the locals tuple, it can be copied directly from a frame pointer offset as needed.

If all arguments to a method are read-only, and no local variables used in the context are captured by a closure, no locals tuple need be allocated.
