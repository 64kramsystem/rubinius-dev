meta_push_2

Byte code: 0x49

Operation

Pushes 2 onto the stack

Format

meta_push_2

Stack

Before After
...
 =>
2
...

Description

Fast push of 2 (two) onto the stack. This is an optimisation applied whenever a literal 2 is encountered by the compiler. It is equivalent to 'push 2', but faster because there is no need to lookup the literal value in the literals tuple.

Source

stack_push(I2N(2));
