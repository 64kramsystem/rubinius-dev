push_block

Byte code: 0x2b

Operation

Pushes a block onto the stack

Format

push_block

Stack

Before After
...
 =>
block
...

Description

Pushes the current block onto the stack. Used when a block passed to a method is used.

Example

def takes_block
  yield      # yields to the block passed to the method, which causes push_block to be called
end

Source
