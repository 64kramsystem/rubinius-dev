attach_method (attach)

Byte code: 0x1c

Operation

Attaches a method definition to an object's singleton class

Format

attach_method name

Stack

Before After
recv
method
...
 =>
method
...

Description

Hooks up a compiled method to an object instance via it's singleton class.

The object the method is to be added to (recv) and the compiled method object (method) are popped from the stack, while the name of the method is an argument to the opcode (name). On return, the compiled method is pushed back onto the stack.

Notes

Class/module methods are handled by add_method.
