passed_arg

Byte code: 0x38

Operation

Test to determine whether an argument was passed

Format

passed_arg idx

Stack

Before After
...
 =>
true | false
...

Description

Checks if the number of arguments passed to a method is greater than the specified argument index idx (0-based), and pushes the result of the test onto the stack.

Source

next_int;
if((unsigned long int)_int < cpu_current_argcount(c)) {
  stack_push(Qtrue);
} else {
  stack_push(Qfalse);
}
