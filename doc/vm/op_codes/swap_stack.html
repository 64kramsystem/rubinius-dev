swap_stack (swap)

Byte code: 0x10

Operation

Swap the top two stack values

Format

swap_stack

Stack

Before After
value1
value2
...
 =>
value2
value1
...

Description

Swaps the top two items on the stack, so that the second item becomes the first, and the first item becomes the second.

Source

t1 = stack_pop();
t2 = stack_pop();
stack_push(t1);
stack_push(t2);
