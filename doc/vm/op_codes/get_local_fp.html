get_local_fp

Byte code: 0x64

Operation

Get the value of a local and push it onto the stack

Format

get_local_fp idx

Stack

Before After
...
 =>
value
...

Description

Retrieves the current value of a local allocated on the stack, and pushes it onto the top of the stack.

Notes

Local variables that are used within a method and not captured are allocated on the stack. This operation retrieves such a local, and places it on the stack.
