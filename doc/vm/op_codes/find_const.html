find_const (find)

Byte code: 0x1b

Operation

Finds a constant

Format

find_const konst

Stack

Before After
ns
...
 =>
const
...

Description

Pops the module or class _ns_ off the stack, and searches within it's namespace for the constant identified by the literal _konst_. If found, it is pushed onto the stack; otherwise, the special value Undefined is pushed instead.

Example

str = "abc"
enum = Enumerable::Enumerator(str, :each_byte)
