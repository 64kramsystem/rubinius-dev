meta_push_neg_1

Byte code: 0x46

Operation

Pushes -1 onto the stack

Format

meta_push_neg_1

Stack

Before After
...
 =>
-1
...

Description

Fast push of -1 (negative 1) onto the stack. This is an optimisation applied whenever a literal -1 is encountered by the compiler. It is equivalent to 'push -1', but faster because there is no need to lookup the literal value in the literals tuple.

Source

stack_push(I2N(-1));
