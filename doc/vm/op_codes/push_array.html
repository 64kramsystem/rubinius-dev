push_array

Byte code: 0x2f

Operation

Unpack the array on the top of the stack into individual items on the stack

Format

push_array

Stack

Before After
[value1, value2, ..., valuen]
...
 =>
value1
value2
...
valuen
...

Description

Removes the array currently on top of the stack, and replaces it with the contents of the array, such that the first item in the array ends up on top of the stack.

Notes

Push array places the contents of the array onto the stack in reverse order to the order in which items on the stack are added to an array by make_array.

j = FIXNUM_TO_INT(array_get_total(t1));

  stack_push(Qnil);
