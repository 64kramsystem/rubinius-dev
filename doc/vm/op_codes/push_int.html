push_int

Byte code: 0x7

Operation

Pushes an integer literal value onto the stack

Format

push_int value

Stack

Before After
...
 =>
value
...

Description

Pushes the value of the integer literal onto the stack.

Notes

Certain common cases (i.e. -1, 0, 1, and 2) are optimised to avoid the lookup of the literal value in the literals tuple.
