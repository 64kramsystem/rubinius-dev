send_super_with_arg_register

Byte code: 0x45

Operation

Call a method on the superclass, passing args in an array plus a block

Format

send_super_with_arg_register mthd

Stack

Before After
blck
args
...
 =>
args
...

Description

Pops a block off the stack, and sends the message mthd to the current objects superclass. The arguments args to the method are contained in an array on the stack, ready to be consumed when the method executes.

Notes

The args register must have previously been set to the count of the number of arguments in args.
