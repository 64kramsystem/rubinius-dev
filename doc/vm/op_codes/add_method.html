add_method (add)

Byte code: 0x1d

Operation

Adds a method to a class or module

Format

add_method name

Stack

Before After
recv
method
...
 =>
method
...

Description

Hooks up a compiled method to a class or module.

The class or module the method is to be added to (recv) and the compiled method object (method) are popped from the stack, while the name of the method is an argument to the opcode (name). On return, the compiled method is pushed back onto the stack.

Notes

Singleton methods are handled by attach_method.
