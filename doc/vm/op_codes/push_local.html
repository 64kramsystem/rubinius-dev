push_local

Byte code: 0x12

Operation

Pushes the value of a local variable onto the stack

Format

push_local local

Stack

Before After
...
 =>
localval
...

Description

Retrieves the current value (_localval) of the referenced local variable (local), and pushes it onto the stack.

Source

next_int; stack_push(tuple_at(state, cpu_current_locals(state, c), _int));
