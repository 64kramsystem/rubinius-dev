send_method

Byte code: 0x28

Operation

Sends a message with no args to a receiver

Format

send_method methodname

Stack

Before After
recv
...
 =>
...

Description

Pops an object off the top of the stack (recv), and sends it the no arg message methodname.

Notes

This form of send is for methods that take no arguments.
