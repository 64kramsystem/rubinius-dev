meta_send_stack_4

Byte code: 0x4d

Operation

Optimised send for performing a method taking four parameters

Format

Description

Pops the receiver recv off the top of the stack, and sends it the method identified by mthd. The four parameters value1 through value4 to be used by the method are left on the top of the stack.
