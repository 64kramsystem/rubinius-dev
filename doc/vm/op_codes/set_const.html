set_const

Byte code: 0x19

Operation

Sets a literal to refer to a constant

Format

set_const lit

Stack

Before After
item
...
 =>
const
...

Description

Pops an object off the stack, and sets the literal referenced in the opcode to refer to the object as a constant. The constant is pushed back onto the stack.

Source

next_int;
t1 = tuple_at(state, cpu_current_literals(state, c), _int);
stack_push(cpu_const_set(state, c, t1, stack_pop(), c->enclosing_class));
