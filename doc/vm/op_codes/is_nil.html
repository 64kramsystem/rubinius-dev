is_nil

Byte code: 0x67

Operation

Return true if value is nil, otherwise false

Format

is_nil

Stack

Before After
value
...
 =>
result
...

Description

Consume the value on the stack, and put the special values true or false depending on whether the consumed value was the special value nil

Source

t1 = stack_pop();
stack_push(t1 == Qnil ? Qtrue : Qfalse);
