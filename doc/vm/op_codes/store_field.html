store_field

Byte code: 0x6

Operation

Store a value into the specified object field

Format

store_field

Stack

Before After
field
value
object
...
 =>

Description

Overwrite the value of a particular field slot with the specified object. The object reference is left on the stack.

Source

t1 = stack_pop();
t2 = stack_pop();
t3 = stack_pop();
SET_FIELD(t3, FIXNUM_TO_INT(t1), t2);
stack_push(t3);
