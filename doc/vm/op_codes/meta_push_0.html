Byte code: 0x47

Operation

Pushes 0 onto the stack

Description

Fast push of 0 (zero) onto the stack. This is an optimisation applied whenever a literal 0 is encountered by the compiler. It is equivalent to 'push 0', but faster because there is no need to lookup the literal value in the literals tuple.
