set_local_fp

Byte code: 0x63

Operation

Updates the value of a stack allocated local variable

Format

set_local_fp idx

Stack

Before After
value
...
 =>
value
...

Description

Uses the value on the top of the stack to update the value of the local variable identified by idx.

The top of the stack is not popped by this operation.

Notes

Local variables that are used within a method and not captured are allocated on the stack. This operation updates such a local from the current top of the stack.
