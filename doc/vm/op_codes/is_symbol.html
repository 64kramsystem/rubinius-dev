is_symbol

Byte code: 0x66

Operation

Return true if value is a Symbol, otherwise false

Format

is_symbol

Stack

Before After
value
...
 =>
result
...

Description

Consume the value on the stack, and put the special values true or false depending on whether the consumed value was of Symbol type

Source

t1 = stack_pop();
stack_push(SYMBOL_P(t1) ? Qtrue : Qfalse);
