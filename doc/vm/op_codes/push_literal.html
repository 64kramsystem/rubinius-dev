push_literal

Byte code: 0xb

Operation

Pushes a literal from the current state onto the stack.

Format

push_literal idx

Stack

Before After
...
 =>
literal
...

Description

The literal identified by the opcode argument (idx) in the current state literals tuple is retrieved and placed onto the stack.

The literals tuple is part of the machine state, and holds all literal objects defined or used within a particular scope.

Source

next_int; stack_push(tuple_at(state, cpu_current_literals(state, c), _int));
