set_class

Byte code: 0x5

Operation

Sets a literal to reference the specified class

Format

set_class

Stack

Before After
literal
clazz
...
 =>
literal
...

Description

Pops a literal reference and a class off the top of the stack, sets the class of the literal to the new class, and then pushes it back onto the stack.

Source

_lit = stack_pop();
t1 = stack_pop();
SET_CLASS(_lit, t1);
stack_push(_lit);
