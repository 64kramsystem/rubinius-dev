Etc.getlogin returns the name of the user who runs this process
Etc.getpwnam returns a Passwd struct instance for the given user
Etc.getpwnam only accepts strings as argument
Etc.getpwuid returns a Passwd struct instance for the given user
Etc.getpwuid only accepts integers as argument
Etc.getgrnam returns a Group struct instance for the given group
Etc.getgrnam only accepts strings as argument
